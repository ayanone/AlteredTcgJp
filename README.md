# AlteredTcgJp

Altered TCG のカードの非公式日本語訳データ・和訳シール作成ツールです。

## 日本語訳データ（AlteredTcgJp.csv）

`AlteredTcgJp.csv` に格納されています。日本語訳は機械翻訳+手作業で作られています。もし間違いを見つけたら、作者（あやのね）までご一報ください。

> **注意:** 非公式訳です。初心者向けに、実際のカードよりも注釈文が多めに付けられています。BGA の日本語訳とは訳語が異なる場合があります。

## 表記ルール

### `[ ]`（角括弧）
カード上ではアイコンであることを示します。

| 表記 | 意味 |
|------|------|
| `[表]` | 手札から場に出た時（手のアイコン）|
| `[ウラ]` | リザーブから場に出た時（カードが光るアイコン）|
| `[両面]` | どこからでも場に出た時（矢印アイコン）|
| `[消耗]` | 消耗アイコン（カードを横向きにする） |
| `[1]`〜`[7]`, `[X]` | コストの数字 |

### `_xxx_`（アンダースコア）
カード上で <u>Fleeting</u> のように**下線付き**で表示される状態を示します。同一の状態は重複して持つことができません。

> CSV では下線付き文字を扱えないため、アンダースコアで代替しています。

### `＜サポート＞`
リザーブにある間だけ機能するサポート能力です。

### `[捨て札]`
リザーブから捨て札にすることで起動するサポート能力であることを示します。

### `[永続]`
リザーブに置かれている限り継続して発動するサポート能力を示します。
## 和訳シールテンプレの使い方

`和訳シールテンプレ.docx` はスタンダードサイズの和訳シールを作るために段組されたWordファイルです。

1. `AlteredTcgJp.csv` から必要なカードのテキストをコピーし、テンプレートに貼り付けます。
2. 印刷時は **A4・フチなし** の設定にしてください。


# セットアップ
コマンドラインツールを使用するためには以下のセットアップが必要です。
1. [Git](https://gitforwindows.org/)をインストールしてください。
2. Git Bash を開き、任意のディレクトリで本リポジトリをgit cloneしてください。
3. [Python](https://www.python.org/) をインストールしてください。
4. プロジェクトのルートに `.env` という名前のファイルを作成し、Google AI Platform の API キーを以下のように記入してください。

```
GEMINI_API_KEY=あなたのAPIキー
```

# コマンドラインツール(Git Bashから実行)

## ユニークカードの登録

カード下部に `BTG-101-U-12345` のような番号が印字されたユニークカードを登録します。画像1枚あたり最大6枚程度を推奨。

```sh
sh register_unique_cards.sh "画像ファイルのパス"
```

画像内のユニークカードが `uniques.csv` に登録されます。機械翻訳の質に問題がある場合は `uniques.csv` をテキストエディタやExcelなどで直接編集できます。

Google AI PlatformのAPIリクエストを1+カード枚数回消費します。（無料プランの場合、1日20回のリクエスト制限があります）

## 和訳シールの作成

カード画像から和訳シールを作成します。カード名が読めるサイズの画像が必要です。画像1枚あたり最大28枚程度を推奨（7×4）。カード上の文字がはっきり見えるほど認識精度は高まります。

```sh
sh export_jp_translation.sh "画像ファイルのパス"
```

カード名（最も重要）、色（陣営）、レアリティ、カード下部のカード番号、コストやスタッツの数値、カードテキストなどから総合的にカードを判断し、`AlteredTcgJp.csv` または `uniques.csv`に記載されているどのカードの画像であるかの確率が高いかを特定し、最も合致しているカードの和訳データをcsvファイルから抽出してpdfファイルに出力します。pdfファイルを印刷して切り取り線で切り取ると和訳シールとして使用できます。

カードの特定精度は約90%で、間違えるケースの大半はC/R/Fのレアリティの間違いです。ユニークカードについてはregister_unique_cards.shによってuniques.csvに登録済みでなければ正しく特定できません。

Google AI PlatformのAPIリクエストを1回消費します。（無料プランの場合、1日20回のリクエスト制限があります）
