# AlteredTcgJp
Altered TCGのカードの日本語訳データです。

# 注意
非公式訳です。特に初心者に見られることを想定して、実際のカードよりも注釈文が多めにつけられています。

BGAはプレイしていないので、BGA上での日本語訳とは訳語が食い違っていると思います。

# 和訳シールテンプレの使い方
和訳シールテンプレは、スタンダードサイズの和訳シールを作るのに都合がいいように段組されたwordファイルです。

必要なカードのテキストをcsvファイルから和訳シールテンプレにコピペしていくことで和訳シールを簡単に作ることができます。

印刷する際には、A4フチなしの設定にしてください。

# 記号の意味
## []
カード上ではアイコンであることを示しています。
## \_xxx\_
カード上で<u>Fleeting</u>といったように下線付きであることを示しています。下線付きで示される<u>状態</u>は、同一の物を重複して持つことができません。

Gitで管理したかったので、データをcsvで保持する必要があり、csvでは下線つきの文字を取り扱えないのでこうなっています。
