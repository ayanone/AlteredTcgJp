# AlteredTcgJp
Altered TCGのカードの日本語訳データです。

OCR機能を実行するためには、お使いのPCにPythonをインストールする必要があります。
.envという名前のファイルを作成し、google ai platformのAPIキーを

GEMINI_API_KEY=THISISYOURYOURAPIKEY

のように登録することで、OCRを用いた機能を使用することもできます。

# 日本語訳データについて
AlteredTcgJp.csvというcsvファイルに格納されています。

非公式訳です。特に初心者に見られることを想定して、実際のカードよりも注釈文が多めにつけられています。

BGAはプレイしていないので、BGA上での日本語訳とは訳語が食い違っていると思います。

# 記号の意味
## []
カード上ではアイコンであることを示しています。
## \_xxx\_
カード上で<u>Fleeting</u>といったように下線付きであることを示しています。下線付きで示される<u>状態</u>は、同一の物を重複して持つことができません。

Gitで管理したかったので、データをcsvで保持する必要があり、csvでは下線つきの文字を取り扱えないのでこうなっています。

# 和訳シールテンプレの使い方
和訳シールテンプレは、スタンダードサイズの和訳シールを作るのに都合がいいように段組されたwordファイルです。

必要なカードのテキストをAlteredTcgJp.csvから和訳シールテンプレにコピペしていくことで和訳シールを作ることができます。

印刷する際には、A4フチなしの設定にしてください。

# OCRユニークカード登録
ユニークカードの画像を用意してください。カード下部のカード番号BTG-101-U-12345といった表記の値を使用しますので、これがつぶれて見えないサイズの画像は対応できません。

一度に処理できるのは画像1枚あたりユニークカード6枚程度だと思います。

コマンドラインで
sh register_unique_cards.sh "画像ファイルのパス付ファイル名"

のコマンドを実行することで、画像ファイルに含まれているユニークカードのカードデータをuniques.csvに登録することができます。（ユニークでなかったりカード番号が読み取れないものは無視されます）

機械翻訳の質に問題がある場合は、uniques.csvを直接テキストエディタなどで編集してテキストを修正できます。

# OCR和訳シール作成

カードの画像を用意してください。カード名を使用しますので、これがつぶれて見えないサイズの画像は対応できません。
一度に処理できるのは画像1枚あたりカード20枚程度だと思います。

ユニークカードの場合はそれに加えてカード下部のカード番号BTG-101-U-12345といった表記の値を使用しますので、これがつぶれて見えないサイズの画像は対応できません。
一度に処理できるのは画像1枚あたりユニークカード6枚程度だと思います。

コマンドラインで
sh export_jp_translation.sh "画像ファイルのパス付ファイル名"
と実行することで、画像に含まれているカードのカード名や色や、ユニークの場合はカード番号からカードを特定し、AlteredTcgJp.csvやuniques.csvに記載されている和訳データを元に和訳シールを作成します。
作成された和訳シールはpdfファイルとして出力されます。
