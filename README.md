# AlteredTcgJp

Altered TCG のカードの非公式日本語訳データ・和訳シール作成ツールです。

## 日本語訳データ（AlteredTcgJp.csv）

`AlteredTcgJp.csv` に格納されています。カラム構成は以下の通りです。

| カラム | 内容 |
|--------|------|
| 年月 | カードセットの発売年月（例: 2024-01）|
| カード番号 | カードの識別番号（例: BTG-052）|
| レアリティ | H / C / R / F / E / U / T |
| 陣営 | Axiom / Bravos / Lyra / Muna / Ordis / Yzmir / Neutral |
| 英語名 | カードの英語名 |
| 日本語名 | カードの日本語名 |
| 能力 | 能力テキストの日本語訳 |
| 訳者コメント | 補足コメント（任意）|

> **注意:** 非公式訳です。初心者向けに、実際のカードよりも注釈文が多めに付けられています。BGA の日本語訳とは訳語が異なる場合があります。

## 表記ルール

### `[ ]`（角括弧）
カード上ではアイコンであることを示します。

| 表記 | 意味 |
|------|------|
| `[表]` | 手札から場に出た時（手のアイコン）|
| `[ウラ]` | リザーブから場に出た時（カードが光るアイコン）|
| `[両面]` | どこからでも場に出た時（矢印アイコン）|
| `[消耗]` | 消耗アイコン（カードを横向きにする） |
| `[1]`〜`[7]`, `[X]` | コストの数字 |

### `_xxx_`（アンダースコア）
カード上で <u>Fleeting</u> のように**下線付き**で表示される状態を示します。同一の状態は重複して持つことができません。

> CSV では下線付き文字を扱えないため、アンダースコアで代替しています。

### `＜サポート＞`
リザーブにある間だけ機能するサポート能力です。

### `[捨て札]`
リザーブから捨て札にすることで起動するサポート能力であることを示します。

### `[永続]`
リザーブに置かれている限り継続して発動するサポート能力を示します。`[永続]` の後に能力テキストが続きます。

## 和訳シールテンプレの使い方

`和訳シールテンプレ.docx` はスタンダードサイズの和訳シールを作るために段組されたWordファイルです。

1. `AlteredTcgJp.csv` から必要なカードのテキストをコピーし、テンプレートに貼り付けます。
2. 印刷時は **A4・フチなし** の設定にしてください。


## セットアップ
コマンドラインツールを使用するためには以下のセットアップが必要です。
1. [Python](https://www.python.org/) をインストールしてください。
2. プロジェクトのルートに `.env` ファイルを作成し、Google AI Platform の API キーを以下のように記入してください。

```
GEMINI_API_KEY=あなたのAPIキー
```

## コマンドラインツール

### ユニークカードの登録

カード下部に `BTG-101-U-12345` のような番号が印字されたユニークカードを登録します。番号がつぶれて読めないサイズの画像には対応できません（画像1枚あたり最大6枚程度）。

```sh
sh register_unique_cards.sh "画像ファイルのパス"
```

画像内のユニークカードが `uniques.csv` に登録されます。機械翻訳の質に問題がある場合は `uniques.csv` をテキストエディタで直接編集できます。

### 和訳シールの作成

カード画像から和訳シールを作成します。カード名が読めるサイズの画像が必要です（画像1枚あたり最大20枚程度）。ユニークカードを含む場合はカード番号も読める必要があります。

```sh
sh export_jp_translation.sh "画像ファイルのパス"
```

カード名・色（陣営）・カード番号（ユニークの場合）からカードを特定し、`AlteredTcgJp.csv` または `uniques.csv` の訳データを使って和訳シールを PDF で出力します。

#### カードの特定方法（OCR）

| 宝石マークの色 | レアリティ |
|---------------|-----------|
| 色なし（白）  | C（コモン）または H（ヒーロー） または T（トークン）|
| 青色          | R（レア）または F（色違いレア） |
| 銅色          | E（エグザルテッド） |
| 金色          | U（ユニーク）|

C と H と T の区別は、それらが同じカード名であることはないのでカード名によって区別できます。

R と F の区別は、カード右上の**旗マークの色**で判断します。同名のCと同じ陣営であればR、異なる陣営であればFです。

| 旗の色 | 陣営 |
|--------|------|
| 茶色   | Axiom |
| 赤     | Bravos |
| ピンク | Lyra |
| 緑     | Muna |
| 青     | Ordis |
| 紫     | Yzmir |
